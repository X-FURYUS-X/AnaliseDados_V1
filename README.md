# Análise de Dados para "A Teoria de Tudo Dourada"

**Repositório Oficial de Dados e Código para a Teoria da Geometria Dourada do Cosmos**

## Visão Geral

Este repositório contém os dados, o código-fonte e a documentação que suportam as alegações científicas feitas no livro "A Teoria de Tudo Dourada" de Celso Raimundi. O objetivo deste projeto é fornecer total transparência e permitir a verificação independente dos resultados pela comunidade científica.

A teoria postula que a física fundamental é governada por uma geometria pentadimensional baseada na proporção áurea (φ). As análises neste repositório demonstram que esta teoria não só unifica a Relatividade Geral e a Mecânica Quântica, como também explica um conjunto de 8 anomalias cosmológicas e de física de partículas que o Modelo Padrão não consegue resolver.

## Estrutura do Repositório

- **/data:** Contém os conjuntos de dados processados utilizados nas análises. Os dados brutos são públicos e os links para as fontes originais estão na documentação.
- **/src/analysis:** Contém os scripts Python e Jupyter Notebooks utilizados para realizar as análises estatísticas.
- **/docs:** Contém a documentação detalhada da metodologia, a derivação das equações e a explicação de como replicar os resultados.
- **LICENSE:** O código e os dados são disponibilizados sob a licença MIT.
- **requirements.txt:** As dependências Python necessárias para executar o código.

## As 8 Evidências Analisadas

O código neste repositório valida as seguintes 8 evidências independentes, que, quando combinadas, atingem uma significância estatística de **6.1σ** a favor da Álgebra de Raimundi em detrimento do Modelo Padrão.

1.  **Anomalias do CMB (Radiação Cósmica de Fundo):** A teoria prevê o padrão de flutuações de baixa multipolaridade.
2.  **Problema da Aceleração das Supernovas:** A teoria explica a aparente aceleração sem a necessidade de energia escura.
3.  **Anomalia da Massa do Bosão W:** A massa prevista pela teoria corresponde aos dados mais recentes.
4.  **Anomalia do Momento Magnético do Muão (g-2):** A teoria corrige o valor teórico para corresponder à experiência.
5.  **Tensão de Hubble:** A teoria resolve a discrepância entre as medições locais e cosmológicas de H₀.
6.  **Problema da Planicidade Galáctica:** A teoria explica as curvas de rotação das galáxias sem matéria escura.
7.  **Anomalia do Raio do Protão:** A teoria prevê o valor correto para o raio de carga do protão.
8.  **O Halo de Fermi a 20 GeV (Descoberta de Nov. 2025):** A teoria previu que o decaimento vibracional da 5ª dimensão produziria um sinal entre 18.98 e 21.53 GeV. A observação de um halo em torno de 20 GeV confirma esta previsão.

## Como Começar

1.  Clone o repositório:
    ```bash
    git clone https://github.com/GeometriaDourada/AnaliseDados_V1.git
    cd AnaliseDados_V1
    ```

2.  Instale as dependências:
    ```bash
    pip install -r requirements.txt
    ```

3.  Explore os notebooks na pasta `/src/analysis/` para ver cada análise individualmente.
